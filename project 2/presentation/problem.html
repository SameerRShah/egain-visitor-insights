The Challenge

Understanding the Problem Statement

Problem Statement

Sales representatives need actionable insights about companies visiting the eGain website to identify potential leads and understand customer behavior patterns.

Objectives

Build a prototype web interface that transforms raw visitor logs into meaningful business intelligence for sales teams.

Key Requirements

Process visitor weblog data for insights
Identify companies from IP addresses
Searchable by various attributes
Live demo accessible for 15 days
GitHub repository with documentation
Presentation explaining the solution

Solution must empower sales reps with actionable visitor insights
