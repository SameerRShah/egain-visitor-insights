eGain Visitor Insights API

This API provides insights about website visitors for eGain sales representatives.

API Endpoints

GET /api/companies

Get a list of companies that have visited the website.

Query parameters:

  • search: Search by company name
  • country: Filter by country
  • sort_by: Field to sort by (default: visit_count)
  • sort_order: Sort order (asc or desc, default: desc)
  • page: Page number (default: 1)
  • per_page: Items per page (default: 10)

GET /api/companies/{company_id}

Get details about a specific company.

GET /api/companies/{company_id}/visits

Get all visits from a specific company.

GET /api/visits

Get a list of all website visits.

Query parameters:

  • company_id: Filter by company ID
  • page_path: Filter by page path
  • device: Filter by device type
  • page: Page number (default: 1)
  • per_page: Items per page (default: 10)

GET /api/statistics

Get statistics about website visitors.
